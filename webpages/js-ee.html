JavaScript Execution Environment

Traditional browser object model (BOM) API (non-standardized)

This object model allows the interaction with the browser itself. It provides access and manipulation of the browser window and perform action that does not exactly affect the page content lik opening a new window, closing a window, adjusting the screen size, etc.


Document object model (DOM) API

This object model allows the programs to edit and manipulate the document structure, content, design, etc. It allows dynamic access to your document.


Application Programming Interface (APIs)

These are constructs that help developers create more diverse functionalities and abstracts complicated code. Multiple APIs are available for certain kinds of tasks and applications.

  3. Web Storage API

  4. Push Notification API
    - it allows web applications to relay and recieve messages from a server even though the web app is not currently active. This allows active communication between the web app and the user. The push notification API is the combination of the Notifications API and the Push API and are usually enabled by a service worker.
