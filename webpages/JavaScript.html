JavaScript

is a well-known scripting language that enables you to create a dynamic website. It is supported by most of today’s popular web browsers such as Google Chrome, Mozilla Firefox, Safari, Edge etc. JavaScript creates an environment for the web page to be more user friendly and interactive. JavaScript also provides extra functionalities through its supported Application Programming Interfaces (APIs).

JavaScript Variable Scope

Global Variable -a variable that is declared outside of a function and can be accessed globally.

Local Variables -a variable that is declared inside of a function and can only be called and accessed within that particular function.

JavaScript Variables and constants

Var -variable that is scoped to the nearest function block

Let -variable that is scoped to the nearest enclosing block

Const - values of constant variables that can’t be changed throughout the program
