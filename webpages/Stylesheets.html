Cascading and HTML Style Sheets

Cascading Style Sheets

The Cascading Style Sheets(CSS) is the language used to specify the presentation of structurally marked up documents. It was developed by Hakon Wium Lee and Bert Bos where it was first released in the year 1996.

Version History of CSS:
  1. CSS Level 1 (W3C Recommendation, December 1996)

  2. CSS Level 2 (W3C Recommendation, May 1998)

  3. CSS Level 2 Revision (W3C Recommendation, June 2011)

  4. CSS Level 3 - This version of the cascading style sheets has a modular approach to CSS development.
