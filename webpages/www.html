World Wide Web

What is WWW

The World Wide Web(WWW) is the collection of web applications, where access to a large collection of resources and information is provided via the Internet. It was founded by Tim Berners-Lee in 1989, about the same time HTTP was developed, and is the most successfull system used today.

How Does the WWW Work

The following are aids to the data transfer of the World Wide Web:

URL - Its URL serves to provide or address the web resources that are available.

IP Address - It is the numerival value associated with a certain device or machine, and is needed for accessibility
