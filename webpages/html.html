What is HTML?

The Hypertext Markup Language(HTML) is one of the two core technologies used for web developing, developed by Sir Tim Berners-Lee in the late 1990 after having wrote the browser and server software.From HTML 1.0, the Hypertext Markup Language has had many updates and improved versions throughout the past decade.
