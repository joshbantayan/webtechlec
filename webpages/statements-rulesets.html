CSS Statements and Rule Sets

CSS Statements

Cascading Style Sheet statements are blocks that begins with a non-space beginning and its ending is at the first closing brace or semicolon, there are two kinds of statements At-rules and Rulesets. At-rules starts consists of an at sign (@) as its beginning, then it is followed by an identifier which is ended by a semi colon. And the Rulesets connects group of declaration of CSS to a condition which is described by a selector.

CSS Rule Sets

CSS rule sets consist of a selector, then a declaration block. A declaration block can be consisted by more than one declaration which are separated by a semi colon. This basically aids in matching the CSS codes with the HTML document, whereas the matched elements are then called subjects.


At-Rules

- @charset - it indicates the character encoding used on the style sheet.
- @import - allows the user to import style rules from other style sheets.
- @namespace - this rule specifies XML namespaces to be used in the CSS.



Nested At-Rules

- @supports - Tests whether or not a web browser supports a feature, if the browser supports the feature, then the browser will apply the style for the said elements.


- @document - Identifies condition for style that is implemented to a specific page.


- @font-face - Allows developers to load custom fonts to be used on a webpage.


- @keyframes - specifies the animation code for a CSS animation sequence.


- @media - allows the application of different styles for different kinds of devices that can access the website.


- @page - it indicates the changes that will happen to the webpage layout when printing the document.
