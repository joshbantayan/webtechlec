CSS Selectors and Rule Precedence

CSS Selectors

Selector Syntax
- sequence of one or more simple selectors separated by combinators with one-pseudo element possibly appended to the last sequence.
- always starts with a type selector or a universal selector.
- cannot contain other type selectors or universal selectors

  1. Selector Group
    - comma-separated list of selectors.
    - represents the union of all elements selected by each of the selectors in the list.

  2. Combinators
    - Descendant combinator (whitespaces, space, tabs, etc.)
    - Child combinators ( > )
    - Sibling combinator:
    Adjacent sibling combinator ( + ), General sibling combinator ( ~ )

  3. Simple Selectors
    - Universal Selector
    - Type Selectors
    - ID Selectors
    - Class Selectors
    - Attribute Selectors
    - Psuedo-Classes
    1. Dynamic Pseudo-Classes
      - Link Pseudo-Classes (:link, :visited)
      - User-Action Pseudo-Class (:hover, :active, :focus)
    2. Target Pseudo-Classes
      :target
    3. Language Pseudo-Classes
      :lang()
    4. UI Element States Pseudo-Classes
      :enabled, :disabled, :checked, :intermediate
    5. Structure Pseudo-Classes
      :root, :first-child, :last-child, :only-child, :Nth-child(), Nth-last-child,
      :first-of-type, :last-of-type, :only-of-type, :nth-of-type(), :nth-last-of-type(), :empty
    6. Negation Pseudo-Classes
      :not()

  4. Pseudo-Elements
    ::first-letter
    ::first-line
    ::before
    ::after

CSS Rule Precedence

  1. Resolution by origin and importance:
    - Transition declaration
    - Important user agent declaration
    - Important user declaration
    - Important override declaration
    - Important author declaration
    - Animation declaration
    - Normal override declaration
    - Normal author declaration
    - Normal user declaration
    - Normal user agent declaration

  2. Resolution by specificity (aka the count/value associated with how a selector is created) of:
    - Count number of ID selectors in the selector.
    - Count number of class selectors attributes selectors, pseudo-classes in the selector .
    - Count number of type selectors and pseudo-elements in the selector.
    - Ignorance of universal selectors.
    Example: concatenation of 3 numbers a-b-c in a number system with a large base

  3. By Order
    - Last one wins

Note:

If two selectors are in line with the same order of priority, then; otherwise, if the selectors are in line with the same order of specificity and priority, then the last one declared is taken.
